# Corporate Strategy Audit: Capital Allocation Efficiency

## Executive Summary
Este proyecto implementa un marco de auditoría automatizado para evaluar la eficiencia en la asignación de capital (CapEx/Opex) entre unidades de negocio (Verticales).

## The Problem: The Egalitarian Trap
Las organizaciones a menudo asignan recursos basándose en inercia histórica o política interna ("Trampa del Igualitarismo"), ignorando los fundamentales económicos subyacentes.

## Methodology
Se aplica el marco de BCG para gestión de portafolio y principios de Microeconomía Gerencial (Besanko):
1. **Ingesta de Datos:** Normalización de métricas de gasto y margen de contribución.
2. **Cálculo de Eficiencia:** Determinación de ROI relativo y discrepancias de asignación (Allocation Mismatch).
3. **Matriz de Decisión:** Clasificación de verticales en:
   - **Stars:** Alto ROI, deben recibir más capital.
   - **Legacy/Dogs:** Bajo ROI, candidatos para desinversión o reestructuración.
